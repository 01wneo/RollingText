# RollingText
Android有上下滚动效果的TextView

标签： 滚动动画 数字动画 文本特效

---

![动画预览][1]

特性
========

- 使用简单，API与TextView类似，setText方法可带有上下滚动的动画
- 支持xml设置android:textSize/android:textColor/android:textStyle等常用属性
- 可高度定制，支持任何单个字符的动画效果
